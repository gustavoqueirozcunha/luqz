# Banco de Referências — Squad Designer

Este arquivo atua como o moodboard intelectual do designer. Antes de iniciar a execução de uma peça, consulte estes guias para calibrar o "olho" para o padrão LUQZ.

## 1. Nível Global de Design Premium
Estes players definem o estado da arte de comunicação visual limpa, focada em conversão e usabilidade. Todo Carrossel ou Reel LUQZ deve se parecer como se tivesse saído destas landing pages.
- [Stripe](https://stripe.com/br) (Tons suaves, tipografia de extremo contraste, uso de afiliação e linhas limpas).
- [Linear](https://linear.app) (Modo noturno perfeito, contraste agressivo, simplicidade extrema focada no tech/premium).
- [Apple](https://www.apple.com) (Espaço negativo como protagonista, produtos/elementos gigantes).

## 3. Direção Estética por Formato

### A. Para Carrosséis
- **Foco:** Sem ruído visual. O leitor está "swipando" (passando para o lado) não para ver a beleza artística do design, mas para buscar a saída da dor dele.
- **Estilo:** *Editorial Minimalista*. Textos gigantes, numeração óbvia no canto, contrastes duros.

### B. Para Covers de Reels
- **Foco:** "A promessa em 3 palavras".
- **Estilo:** *Typography Bold*. Nada de capas complexas. Fundo escurecido da cena do próprio vídeo + Texto massivo no centro focando na principal curiosidade do reel.

### C. Para Stories
- **Foco:** Baixa fricção. Design *nativo*.
- **Estilo:** Quando formos fabricar artes para stories, elas devem simular os widgets do Instagram (caixas arredondadas, tipografia nativa), para não parecer um "panfleto empurrado".
