## Como Usar Este Índice

1. Para documentar um fluxo novo: executar Squad luqz-n8n com `/opensquad run luqz-n8n`
2. Para atualizar documentação de fluxo existente: fornecer nome do fluxo e iniciar novo ciclo
3. Para auditar todos os fluxos não documentados: fornecer lista de workflows do N8N e iniciar ciclo em batch
