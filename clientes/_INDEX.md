## Como usar este arquivo

1. **Novo cliente** → copiar `clientes/_TEMPLATE_CLIENTE/`, preencher e adicionar linha aqui
2. **Mudança de fase** → atualizar a coluna "Fase Atual" e "Semana"
3. **Bloqueio** → trocar status para 🔴 e adicionar nota na seção de detalhes
4. **Encerramento** → trocar status para ⚫ e mover para seção "Clientes Encerrados" (criar se necessário)
