# Direção Visual — LEVITÁ

> **Instrução:** Este documento define a percepção instantânea que a marca deve gerar no espectador. O preenchimento dele é obrigatório antes da criação da estampa visual.

## 1. Estilo e Percepção da Marca

- **Sensação Desejada:** Leve, Acolhedor, Premium, Confiável, Natural, Cuidadoso
- **O que a marca transmite em 3 segundos:** "Essa comida foi feita com cuidado de verdade — não é qualquer marmita"
- **Arquétipo visual:** Natural premium — pensa Farmácia Aveda, não academia bright. Pensa Natura premium, não Bio Extratus genérico.

## 2. Termômetro Visual

- [x] **Moderno / Clean:** Forte uso de espaços em branco, respiro, tipografia elegante, pouca textura. Este é o registro principal da LEVITÁ.
- [ ] ~~Conservador / Corporativo~~ — muito frio para uma marca de comida afetiva
- [ ] ~~Bold / Agressivo~~ — remete a fast food e academia, contrário ao posicionamento

## 3. Matriz de Referências

- **Norte 1 (Estética):** Apple — espaço negativo como protagonista, produto em destaque, minimalismo sofisticado
- **Norte 2 (Tipografia):** Stripe — tipografia de extremo contraste, elegância sem frieza
- **Norte 3 (Nicho alimentar):** @healthlifecongelados (Campinas) — referência operacional de marmita saudável com produção real como conteúdo

## 4. O Que EVITAR Absolutamente

- 🛑 Nunca usar fundos vermelhos ou amarelos — remetem a fast food imediatamente
- 🛑 Nunca usar tipografia pesada/condensed/black — oposto da leveza da marca
- 🛑 Nunca usar banco de imagens genérico de comida — foto tem que ser do produto real
- 🛑 Nunca exceder 3 cores por peça — paleta contida é lei
- 🛑 Nunca misturar identidade visual do Dona Gê com a LEVITÁ — marcas separadas
- 🛑 Nunca usar elementos "natureba" clichê (folha de alface recortada, suco verde, raio de sol) — pastiche de marca saudável sem personalidade
