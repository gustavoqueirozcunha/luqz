## 5. MAPEAMENTO PARA O SQUAD

### Por tipo de asset — O que cada agente faz

#### CARROSSEL EDUCATIVO
| Agente | Ação |
|--------|------|
| **Leo** | Define conceito (1 linha), sistema de slides (hierarquia, progressão), instrução de fundo, paleta por slide, instrução para Ivo |
| **Ivo** | Gera visual de capa do carrossel via **Claude Design** (fundo Verde LEVITÁ + tipografia hero); gera elemento gráfico de suporte (textura, linha) |
| **Caetano** | Monta cada slide no Canva: importa assets de Ivo, aplica copy de Clara slide a slide, respeita grid 1080x1350 com margem 80px |
| **Nina** | Valida: mesmo template em todos os slides, hierarquia H1>H2>body>CTA, contraste WCAG AA, máx 3 cores por peça |

#### STATIC FEED
| Agente | Ação |
|--------|------|
| **Leo** | Define conceito da peça, hierarquia (headline + subheadline + CTA), instrução de visual de fundo ou imagem produto |
| **Ivo** | Gera fundo visual via **Claude Design** (composição minimalista, fundo sólido Verde ou Creme) OU importa foto real do produto (quando disponível) |
| **Caetano** | Monta no Canva: aplica tipografia Cormorant (headline) + DM Sans (body + CTA), overlay se imagem, marca discreta no canto |
| **Nina** | Valida: peso tipográfico mínimo 500, espaço em branco intencional, CTA distinguível, sem estética "Canva padrão" |

#### REELS COVER
| Agente | Ação |
|--------|------|
| **Leo** | Define promessa de 3–6 palavras, instrução de background (frame do vídeo ou cena de cozinha), hierarquia |
| **Ivo** | Gera cena de cozinha/preparo via **Claude Design** (quando frame real não disponível); aplica overlay escuro Verde LEVITÁ 60–80% |
| **Caetano** | Monta no Canva: texto hero centralizado Cormorant Bold, zona segura 250px topo + 300px base, logo discreto |
| **Nina** | Valida: texto legível em 3 segundos, máx 2 cores além do fundo, promessa clara, sem elementos desnecessários |

#### STORIES (sequência)
| Agente | Ação |
|--------|------|
| **Leo** | Define sequência de 3–5 frames: narrativa, ordem, instrução visual de cada frame |
| **Ivo** | Gera elementos visuais via **Claude Design** (backgrounds sutis, glassmorphism leve); nano-banana-2 se tiver sujeito consistente (Gerlane em cozinha) |
| **Caetano** | Monta no Canva: simula widgets nativos do Instagram, texto curto e centrado, CTA no frame final |
| **Nina** | Valida: consumível em 3 segundos por frame, máx 3 elementos por tela, sequência narrativa coerente |

#### KIT NUTRICIONISTAS (1 página)
| Agente | Ação |
|--------|------|
| **Leo** | Define hierarquia do documento: logo > quem somos > menu resumido > diferenciais > contato |
| **Ivo** | Gera visual de capa e elementos gráficos via **Claude Design** (header premium, dividers, ícones thin stroke) |
| **Caetano** | Monta no Canva: formato A4 ou 1080x1920, tipografia sistema LEVITÁ, exporta PDF |
| **Nina** | Valida: leitura rápida (5 segundos identificar proposta), brand compliance total, sem elementos genéricos |

## 7. PADRÕES VISUAIS LEVITÁ — Regras do Squad

### Sistema visual obrigatório (não negociável)

```
SISTEMA VISUAL LEVITÁ v1.0

Paleta (máx 3 cores por peça):
  Verde LEVITÁ: #3B6B4A — fundo principal, logo, destaques
  Creme: #F5EEE4 — fundo claro, texto sobre verde, respiro
  Cobre: #C47950 — accent APENAS em CTA e destaques pontuais
  Grafite: #1E1E1E — texto principal em fundos claros
  Verde Claro: #A8C5A0 — subtextos sobre verde, separadores
  Bege Soft: #E8DDD1 — cards, camadas secundárias
  (Regra de ouro: Verde LEVITÁ + Creme = combinação principal)

Tipografia:
  Display/Capa: Cormorant Garamond — Semibold Italic (logo) | Bold (H1)
  Corpo/Detalhes: DM Sans — Medium (H2) | Regular (body) | Light (caption)
  Tracking Display: +20 a +40
  Line-height: 140% corpo | 120% títulos

Tamanhos (formato 1080x1350px):
  H1 (capa): 90–120px Cormorant Garamond Bold
  H2 (subtítulo): 44–56px DM Sans Medium
  Body: 32–38px DM Sans Regular
  Caption: 22–26px DM Sans Light
  Margem: mínimo 80px em todas as bordas

Regras absolutas:
  🛑 Nunca: fundo vermelho ou amarelo
  🛑 Nunca: tipografia condensada ou pesada
  🛑 Nunca: imagem de banco genérica (foto deve ser do produto real)
  🛑 Nunca: mais de 3 cores por peça
  🛑 Nunca: misturar identidade LEVITÁ com Dona Gê
  🛑 Nunca: gradiente radial (linear apenas)
  🛑 Nunca: shadow dura (blur mín 40px, opacidade máx 12%)
  🛑 Nunca: elementos "natureba" clichê (folha de alface picada, suco verde, raio de sol)
```

### Consistência entre peças do mesmo lote

- 1 template base por tipo de formato — sem variação de template dentro do carrossel
- Background principal consistente: Verde LEVITÁ ou Creme — não alternar por slide sem intenção
- Logo sempre no canto superior esquerdo (padrão) ou centralizado apenas em capas institucionais
- CTA sempre em Cobre (#C47950) — diferenciado visualmente do restante

### Variações controladas

| Elemento | Variação permitida | Variação proibida |
|----------|-------------------|------------------|
| Fundo | Verde LEVITÁ ↔ Creme | Qualquer outra cor como fundo dominante |
| Accent | Cobre pontual (CTA) | Cobre dominando mais de 20% da peça |
| Tipografia | Cormorant ↔ DM Sans | Qualquer outra família |
| Imagem | Foto real produto / cena cozinha | Banco de imagem genérico |
| Gradiente | Linear (Verde → Creme) | Radial em qualquer situação |
