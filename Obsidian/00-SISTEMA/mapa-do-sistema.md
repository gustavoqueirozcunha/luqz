## Estrutura de Pastas (raiz)

```
luqz/
├── CLAUDE.md               ← Lei central. Todo agente obedece.
├── clientes/               ← Contexto + produção de cada cliente
├── Produtos/               ← Metodologia e especificações comerciais
├── squads/                 ← Agentes + pipelines de execução
├── skills/                 ← Capacidades integradas (Canva, IA, etc)
├── workflows/              ← Automações N8N
├── docs/                   ← Documentação operacional
├── Obsidian/               ← Cérebro autoral (você está aqui)
├── dashboard/              ← Escritório virtual (monitoramento)
└── _opensquad/             ← Framework de geração de squads
```
