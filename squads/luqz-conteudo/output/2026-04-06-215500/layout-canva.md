# Layout de Carrossel — O gargalo oculto da receita
**Modo:** Instrução Detalhada

## Template Único Recomendado
- **Nome/Categoria:** Carrossel Minimalista Empreendedor Dark (Estilo SaaS / Stripe / Linear)
- **Justificativa:** O template garante forte contraste, minimalismo sem distrações e uma progressão de leitura muito limpa, condizente com a seriedade para o ICP do B2B que busca resolver gargalos financeiros.
